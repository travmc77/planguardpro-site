How to Submit Your Plans

Follow these steps to ensure your screening gets started without delays.

1

Separate Files by Discipline

Split your plan set into individual PDF files by discipline. Name each file using your HCAI application number (if available) or your firm's project number:

[HCAI App Number]_[Discipline].pdf
S241906-33-00_Architectural.pdf S241906-33-00_Structural.pdf S241906-33-00_Electrical.pdf S241906-33-00_Mechanical.pdf S241906-33-00_Plumbing.pdf S241906-33-00_FireLifeSafety.pdf
If no HCAI number is available, use your firm's project number: [FirmProjectNumber]_[Discipline].pdf
2

Identify Submission Type

Let us know which type of submission you're screening for:

New Submission Initial plan set for HCAI review (BC0)
ACD (Amended Construction Document) Revisions to previously approved construction documents
DSI (Deferred Submittal Item) Post-approval deferred items for separate review
3

Upload Your Files

After payment, you'll be directed to our secure upload form. Upload your discipline PDFs, supporting documents, and project details ‚Äî files go directly to our Google Drive. Include:

  • Facility name
  • HCAI application number (if available)
  • Submission type (New / ACD / DSI)
  • Total sheet count
  • Project-specific TIO (Title, Intent & Objectives)
  • Project narrative / scope description
  • Any other supporting documents being submitted to HCAI (e.g., functional program, equipment lists, hazardous materials inventory, geotechnical report)
  • Any specific areas of concern
